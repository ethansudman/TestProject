# TestProject
This project demonstrates how to use C# and vanilla JavaScript to display a directory structure as a TreeView and then upload and download files from it.

It was developed in Visual Studio Community 2026. It can be run simply by pressing F5.

The project will create a directory called "BrowsableDirectory" in the same location as the executable and populate it with some sample data, and it will display the contents of that directory in the TreeView. You can upload files to that directory using the "Upload File" button, and you can download files by clicking on them in the TreeView.
